# 🎨 Artistic Image Studio

A MATLAB App Designer application for **Non-Photorealistic Image Rendering (NPR)** and **Artistic Style Transfer** using Digital Image Processing techniques.

## 📖 Overview

Artistic Image Studio is an interactive MATLAB-based application developed as a Digital Image Processing (DIP) course project. The application allows users to upload images and transform them into various artistic styles inspired by traditional art forms.

The project demonstrates the practical implementation of core Digital Image Processing concepts such as edge detection, spatial filtering, intensity transformations, bilateral filtering, and color quantization through an intuitive graphical user interface.

---

## ✨ Features

* Upload JPG, JPEG, and PNG images
* Real-time artistic image transformation
* User-friendly MATLAB App Designer interface
* Side-by-side comparison of original and processed images
* Fast image processing workflow
* Multiple artistic rendering styles

---

## 🎭 Artistic Filters

### ✏️ Pencil Sketch

Transforms photographs into hand-drawn pencil sketches using:

* Grayscale conversion
* Image inversion
* Gaussian blur
* Dodge blending

### 🖋️ Ink Outline

Generates clean black-ink drawings using:

* Canny edge detection
* Morphological operations
* White canvas overlay

### 🎨 Oil Painting

Creates a realistic painting effect using:

* Bilateral filtering
* Color quantization
* Median filtering
* Edge sharpening

### 🖤 Charcoal Sketch

Produces charcoal-style artwork through:

* Texture extraction
* Shadow blending
* Contrast enhancement

### 🌈 Pop Art

Creates bold poster-style images using:

* Color quantization
* Edge outlining
* Contrast boosting

## 🏗️ Application Workflow

1. Upload an image.
2. Select an artistic filter.
3. The image is processed using DIP algorithms.
4. The transformed image is displayed instantly.
5. Compare original and processed results side-by-side.
